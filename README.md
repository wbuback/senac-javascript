# Taskschool — Projeto de Exemplos para Sala de Aula

Projeto JavaScript progressivo para ensinar lógica de programação e back-end.
Cada etapa é um arquivo `.js` independente que pode ser executado com `node`.

## Como usar

```bash
node etapa1.js
node etapa2.js
# ... e assim por diante
```

## Sequência de Ensino

| Arquivo     | Conteúdo                                    |
|-------------|---------------------------------------------|
| etapa1.js   | Algoritmos e pensamento computacional        |
| etapa2.js   | Variáveis, constantes, tipos e operadores   |
| etapa3.js   | Condicionais: if / else / switch             |
| etapa4.js   | Loops: for, while, forEach, map, filter     |
| etapa5.js   | Funções: declaration, expression, arrow     |
| etapa6.js   | Objetos e arrays avançados                  |
| etapa7.js   | Orientação a Objetos: classes e herança     |
| etapa8.js   | Async/Await e Promises                      |
| etapa9.js   | Módulos e organização de projeto            |

## Pré-requisitos

- Node.js versão LTS instalado
- Verificar: `node --version`

## Projeto base

Todos os exemplos usam o **Taskschool** (gerenciador de tarefas) como contexto,
tornando o aprendizado consistente ao longo de todas as etapas.
